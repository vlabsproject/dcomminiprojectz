Instruction

  1. First turn on the 8 Bit variable data generator using power switch, which helps to generate the data signal from given data.

  2. Change the orientation of Data switch as per as you data, for 0 toggle it to up or vice versa.

  3. Check tha data again from S0 to S7 bit and set the amplitude and frequency of message signal.

  4. Click on RUN button to generate the ASK signal.

  5. Observe the graph and download it from right top corner of graph in png, jpeg, pdf format.

  6. To decode the ASK signal click the decode button shown below.
